Calculadora de Empréstimos

Desafios / Conceito por trás / Aprendizado

Desafio:

  • Utilizar DOM
  • Projeto inteiro em JavaScript
  • Aprender a usar métodos de Math

Conceito por trás

Idéia era juntar conhecimento adquirido de tutoriais de JavaScrip. pegar uma idéia e tentar desenvolver, buscando informações pela internet. Idéia, também, utilizar o projeto inteiramente em inglês a fim de praticar e desenolvimento pessoal.

Aprendizado

Projeto relativamente simples, mas bastante desafiador. Tentei utilizar boas praticas no desenvolvimento tanto no JavaScript, quanto no HTML e CSS. CSS feito com Lib Materialize. Primeira vez utilizado, também me trouxe o desafio de fazer a leitura correta da documentação e me trouxe experiência para lidar com JavaScript manipulando todo documento, utilizando métodos de matemática para fazer os cálculos e tornar o app o mais real possível.

RGB Game - Escolha a cor correta

Desafios / Conceito por trás / Aprendizado

Desafio:

  • Projeto inteiro em JavaScript
  • Aprender a gerar cores randomizadas
  • Criar níveis de difículdades para o jogo

Conceito por trás

Neste projeto, a idéia era criar um jogo a fim de praticar a geração de cores aleatórias. Processo basicamente feito definindo valores em RGB aleatórios através de métodos de geração de números do JS. Apenas uma forma descontraída de praticar e ao mesmo tempo se divertir.

Aprendizado

Tentei o mais agradável visualmente possível. Usar o conhecimento para criar um mini-game, praticando e errando. Mais um projeto feito em Inglês a fim de praticar o uso da língua na programção. Boas praticas adodatas, sempre tentando tornar a aplicação mais funcional possível. Tentei definir bons padrões para que, no futuro, se necessário, fosse fácil a identificação e aperfeiçoamento do código.

Task-List - Crie sua lista de tarefas

Desafios / Conceito por trás / Aprendizado

Desafio:

  • Projeto inteiro em JavaScript
  • Criar uma Task List
  • Tornar a Task List dinâmica
  • Usar Local Storage
  • Usar métodos de filtragem nos itens

Conceito por trás

De longe, o mais divertido e difícil de ser feito. Desafio enorme, demorou alguns bons dias para ser feito, mas com muito gosto saiu. Nesse projeto a ideia era praticar conceitos básicos de JavaScript. Não foi utilizado nada muito avançado, para filtragem, utilizado condicionais com métodos de identificação. Para limpar, funções que fazem esse trabalho, inteiramente interligado com DOM.

Aprendizado

Mais um projeto feito com JavaScript puro, sem nada fancy. A ideia era bater com a cara na parede, aprender, e evoluir. Objetivo concluído. Como apresentado no vídeo, projeto simples, bem estruturado e com CSS feito utilizando Lib Materialize.
